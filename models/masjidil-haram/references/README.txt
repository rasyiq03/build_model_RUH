Put the reference files here, sorted by type (see ../AGENTS.md §R):
  aerial/    top / satellite views (footprint, Mataf ring, positions)
  exterior/  facades, minarets, domes, gates
  interior/  prayer halls, columns, Mataf levels, Mas'a corridor
  section/   cutaways showing how floors stack
  detail/    close-ups: Kaaba/kiswah, columns, gates, lamps
  models/    the rough 3D model (grand+mosque+mecca+3d+model) — cross-check only
  _clarify/  (auto) annotated images + notes + uploads from the ref-clarify skill

OSM_REFERENCE.md in this folder is the XY-layout source of truth.
The Drive renders (wire_masjid_*) typically go in exterior/ or models/.
